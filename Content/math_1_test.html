แบบทดสอบคณิตศาสตร์สำหรับฟิสิกส์

เลือกคำตอบที่ถูกต้องที่สุด

  1. กำหนดให้ $$N=\frac{(2\times 10^4)(3\times 10^{-12})}{10^{-6}}$$ จงหาค่าของ N
    ( a ) $$6\times 10^{-22}$$ ( b ) $$6\times 10^{-2}$$ ( c ) $$6\times 10^5$$ ( d ) $$6\times 10^2$$ ( e ) $$6$$

  2. $$\sqrt{81x^4y^{10}}$$ มีค่าเท่ากับข้อใด
    ( a ) $$9xy^2$$ ( b ) $$3x^2y^5$$ ( c ) $$9x^4y^{10}$$ ( d ) $$9x^2y^5$$ ( e ) $$27(xy)^7$$

  3. ถ้า Y เป็นปฏิภาคกลับ กับ x โดยที่ Y=28 แล้ว x=9 และถ้า Y=42 แล้ว x = ?
    ( a ) 13.5 ( b ) 6 ( c ) 14 ( d ) 49 ( e ) 12

  4. ภาพที่แสดงต่อไ่ปนี้เป็นกราฟระหว่าง x กับ y
    ถามว่า x มีความสัมพันธ์กับ y อย่างไร
    ( a ) เป็นปฏิภาคกลับ ( b ) เป็นปฏิภาคโดยตรง ( c ) เป็นปฏิภาคร่วมกัน

  5. กำหนด P เป็นปฏิภาคโดยตรงกับ T โดยเมื่อ P = 10 แล้ว T = 8 ทีนี้ ถ้า T = 36 แล้ว P = ?
    ( a ) 32 ( b ) 45 ( c ) 12 ( d ) 40 ( e ) 64

  6. $$x^2 + 5x + 6 = 0$$ แล้ว $$x =\ ?$$
    ( a ) -2,-3 ( b ) 5,1 ( c ) 2, 3 ( d ) -1,-5 ( e ) 7,-1

  7. $$\sqrt{242.75}$$ มีค่าเท่าใด (ตอบข้อที่ใกล้เคียงที่สุด)
    ( a ) 13 ( b ) 15.9 ( c ) 15.6 ( d ) 15.2 ( e ) 15.0

  8. ถ้า $$w=\frac{mv^2}{2}+U$$ และ $$U=-\frac{e^2}{R}$$ และ $$\frac{mv^2}{R}=\frac{e^2}{R^2}$$ แล้ว $$w$$ มีค่าเป็นไปตามข้อใด
    ( a ) $$-\frac{e^2}{R}$$ ( b ) $$\frac{e^2}{2R}$$ ( c ) $$-\frac{e^2}{2R}$$ ( d ) $$\frac{e^2}{R^2}$$ ( e ) $$\frac{e}{R}$$

  9. $$(\frac{2}{3}+\frac{3}{5})$$ มีค่าเท่ากับ ข้อใด
    ( a ) $$\frac{6}{8}$$ ( b ) $$\frac{3}{4}$$ ( c ) $$\frac{2}{5}$$ ( d ) $$\frac{19}{15}$$ ( e ) $$\frac{5}{8}$$

  10. กำหนดให้ $$d=\frac{M}{V}$$ เมื่อ d หมายถึง ความหนาแน่น สมการนี้บอกอะไรแก่เรา
    ( a ) ความหนาแน่น เป็นปฏิภาคโดยตรงกับทั้ง มวล และ ปริมาตร
    ( b ) ปริมาตร เป็นปฏิภาคโดยตรงกับทั้ง มวล และ ความหนาแน่น
    ( c ) มวล และ ปริมาตร เป็นปฏิภาคกลับ ซึ่งกันและกัน
    ( d ) ปริมาตร และ ความหนาแน่น เป็นปฏิภาคโดยตรง ซึ่งกันและกัน
    ( e ) ความหนาแน่น และ ปริมาตร เป็นปฏิภาคกับ ซึ่งกันและกัน
